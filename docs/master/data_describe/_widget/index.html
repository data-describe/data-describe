data_describe._widget¶

class data_describe._widget.BaseWidget(compute_backend=None, viz_backend=None, **kwargs)¶

Bases: abc.ABC

Interface for collecting information and visualizations for a feature.

A “widget” serves as a container for data, diagnostics, and other outputs (i.e. DataFrames, plots, estimators etc.) for a feature in data-describe.

abstract show(self, viz_backend=None)¶

Show the default output.

Assembles the object to be displayed by _repr_html_. This should respect the viz_backend, if applicable.

Raises
